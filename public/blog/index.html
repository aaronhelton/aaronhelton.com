The Real Curse of Strahd

In pursuit of an idea1, I decided to run the original 1983 Ravenloft module. You know, the AD&D module that introduced us to the lord of the Domain of Dread, Strahd von Zarovich. The setup was simple enough: I proposed to run the adventure using Old School Essentials characters and mechanics, and it would take a session or two. Or so I thought.
